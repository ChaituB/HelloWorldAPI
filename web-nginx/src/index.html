API Access logs

This is a static, Nginx served application which calls Hello-World API (which is on node docker container) and displays the logs for /v1/logs and /v1/hello-world/logs endpoints. It uses JavaScript Ajax calls to call the API and display the results.

  • Hello-World Logs

    This page asynchronously calls /v1/hello-world/logs API and displays the access log (IP, and Unix Timestamp) for /v1/hello-world endpoint.

    VISIT /v1/hello-world/logs

  • All Access Logs

    This page asynchronously calls /v1/logs API and displays the access log (Endpoint, IP, and Unix Timestamp) for /v1/* endpoints.

    VISIT /v1/logs

Any call to the page other than the above said two redirects to this (HomePage).
