Hello-World REST API

This is a stateless RESTlike Hello-World API which returns {"message": "hello world"}.

Public EndPoints

  • GET /v1/hello-world

    Returns JSON object {"message": "hello world"}


Private EndPoints

  • GET /v1/logs

    Returns JSON object {"logs": [{"ip": "", "timestamp":""}]}

  • GET /v1/hello-world/logs

    Returns JSON object {"logset": [{"endpoint": "", "logs": [{"ip": "", "timestamp":""}]}]}


Other EndPoints

  • For all the other endpoints and http methods, it returns JSON error object {"error": "resource not found"}
